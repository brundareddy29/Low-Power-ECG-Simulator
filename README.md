 Designed and implemented a low-power ECG monitoring system using the AD8232 analog front-end module and
Arduino Uno for real-time cardiac signal acquisition.
– Captured PQRST waveforms using a three-electrode configuration, enabling accurate representation of heart activity.
– Applied signal conditioning and moving average filtering techniques to reduce noise and improve ECG signal quality.
– Developed a cost-effective solution for wearable healthcare, remote patient monitoring, and biomedical education
